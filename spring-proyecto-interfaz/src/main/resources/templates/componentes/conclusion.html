Este proyecto tiene como objetivo desarrollar y implementar un sistema integral de registro hospitalario para mejorar la eficiencia en la gestión de pacientes, la precisión en el almacenamiento de datos médicos y la coordinación entre distintos departamentos. La solución propuesta busca cumplir con los estándares regulatorios actuales y optimizar los procesos internos del hospital.
